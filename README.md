# Credit Risk Insight Engine

An end-to-end analytics system that turns raw loan-performance data into
self-serve portfolio risk insights — combining a SQL warehouse, a Power BI
risk dashboard, an interpretable default-risk model, and a natural-language
query layer that lets non-technical stakeholders ask questions in plain English.
